## 技术栈 react + express + mongoose


### 运行本项目：
    git clone https://github.com/lindazhang102/exercise-tracker.git
    cd backend
    npm install
    node server.js

    切窗口：
    cd exercise-tracker
    npm install
    npm start
### 本项目使用的是云端 MongoDB 服务
    链接：<https://www.mongodb.com/cloud/atlas>
    需要自己在项目中配一下自己的 MongoDB 服务链接，配置文件地址：backend/.env
